# 部署手册：AI辅助个人智能记账系统

## 方式一：Vercel（推荐）

1. 打开 https://vercel.com/new
2. Import 仓库 `absAline/bookkeeping`
3. 框架选 Astro
4. Build: `npm run build`
5. Publish: `dist`
6. Deploy

## 方式二：Netlify

1. 打开 https://app.netlify.com/start
2. 连接 GitHub 仓库
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Deploy

## 方式三：本地运行

```bash
cd bookkeeping
npm install
npm run dev       # 开发模式 http://localhost:4321
npm run build     # 构建
npm run preview   # 预览构建结果
```

## 数据说明

所有数据存储在浏览器 localStorage 中，部署无需数据库。如需持久化，可自行接入 Supabase 或 Firebase。

## 后台编辑器

```bash
cd admin
npm install
node server.js   # 端口 4324
```
