# AI辅助个人智能记账系统 - 操作手册

## 环境要求
- Node.js 22.12+
- npm 10+

## 安装步骤
```bash
cd bookkeeping
npm install
```

## 运行方式
```bash
# 开发模式
npm run dev
# 访问 http://localhost:4321

# 构建生产版本
npm run build

# 预览生产版本
npm run preview

# 启动后台编辑器
cd ../admin && npm start
# 访问 http://localhost:4324
```

## 功能详解

### 概览页面
- **收入/支出/结余卡片**：自动汇总当月数据，结余为负时显示红色
- **月筛选器**：切换查看不同月份的概览
- **支出排行**：显示当月各分类支出金额及占比进度条
- **AI 分析建议**：至少 3 条支出后自动生成消费分析

### 记账操作
1. 点击「+ 记一笔」打开表单
2. 选择类型（支出/收入）
3. 输入金额和备注
4. 选择分类（或输入备注后 AI 自动识别）
5. 选择日期，点击「保存」
6. 编辑：点击账单右侧「编辑」按钮
7. 删除：编辑模式下点击「删除」

### 账单列表
- 按月份切换
- 按支出/收入筛选
- 按备注关键词搜索
- 编辑/删除记录
- 「清空本月数据」批量删除

### 图表分析
- **收支趋势**：折线图展示每日收入和支出
- **支出分类**：环形图展示各分类占比
- 切换月份自动刷新

### 设置页面
- **导出 JSON**：下载全部数据的 JSON 文件
- **导出 CSV**：下载全部数据的 CSV 文件
- **导入 JSON**：上传之前导出的 JSON 文件恢复数据
- **分类管理**：支持自定义分类
- 数据存储在浏览器 localStorage，清除缓存会丢失

## 常见问题
| 问题 | 解决方案 |
|------|----------|
| 页面空白或报错 | 确认 `npm install` 完成且使用 Node.js 22+ |
| 端口 4324 被占用（后台编辑器） | 修改 `admin/server.js` 中的端口号或关闭占用程序 |
| 数据丢失 | 定期导出 JSON 备份 |
| 图表不显示 | 检查网络是否能访问 Chart.js CDN |
| AI 分类不准确 | 补充关键词或手动选择分类 |

## 维护说明
- 所有数据存储在浏览器 localStorage，无后端数据库
- 数据导出为 JSON 格式，可随时备份和恢复
- 如需迁移数据：导出 JSON → 在新浏览器导入
- 分类关键词映射在 `src/utils/store.js` 的 `guessCategory` 函数中
